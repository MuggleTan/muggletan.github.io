开启idea的自动编译

打开顶部工具栏 File -> Settings -> Default Settings -> Build -> Compiler 然后勾选 Build project automatically

开启idea的热部署策略

顶部菜单- >Edit Configurations->SpringBoot插件->目标项目->勾选热更新
