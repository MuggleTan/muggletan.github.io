Java中的栈和队列

发表于 2024-08-01 更新于 2025-04-15 阅读次数：

栈

先进后出（FILO）

  • push: 将元素压入栈顶。
  • pop: 移除并返回栈顶元素。
  • peek: 返回栈顶元素但不移除它。

队列

先进先出（FIFO）

  • offer: 将元素添加到队列尾部。
  • poll: 移除并返回队列头部的元素。
  • peek: 返回队列头部的元素但不移除它。

双端队列（Deque）

双端队列（Deque, Double Ended Queue）是允许在两端进行插入和删除操作的队列。Java中的ArrayDeque类是Deque接口的一个实现。
